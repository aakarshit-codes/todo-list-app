### Description

Build a simple **Todo List App** where users can add, edit, and delete tasks. Use objects and arrays to manage the state of the tasks, and save the list in **local storage** so it persists between page reloads.

### Topics Covered

- Objects and Arrays
- DOM Manipulation
- Closures and Scopes
- Local Storage (using objects and arrays)

### Features

- Add new tasks.
- Mark tasks as completed
- Delete tasks.
- Store tasks in the browser's **localStorage**
